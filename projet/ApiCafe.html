Api Cafe

L'application contient une API faite en MVC et testée sur Postman avec la possibilité de se connecter avec un jeton et d'autres routes.

Interface React

Ensuite, création d'un projet React pour interroger l'API et afficher dans un tableau les informations des dosettes ainsi que de pouvoir les filtrer, modifier, ajouter ou supprimer si l'on est administrateur, et une page d'authentification.
