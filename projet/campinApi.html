Description

Camping API est une application Spring Boot qui fournit une API RESTful pour gérer les données liées au camping. Elle permet l'authentification des utilisateurs, la gestion des comptes, et la persistance des données.

Fonctionnement

L'application permet aux utilisateurs de s'authentifier, de gérer leurs comptes, de réserver des créneaux pour des activités, et de s'inscrire ou se désinscrire d'activités.


Ce qu'il y a dans l'application :


    Authentification : Les utilisateurs s'authentifient via un point de terminaison POST /auth/login, qui retourne un jeton JWT pour les requêtes sécurisées.


    Gestion des Comptes : Les comptes peuvent être récupérés, bloqués, et gérés via des points de terminaison spécifiques, nécessitant des rôles d'utilisateur appropriés (client ou admin).


    Gestion des Créneaux : Les créneaux horaires peuvent être récupérés pour planifier des activités.


    Gestion des Inscriptions : Les utilisateurs peuvent s'inscrire ou se désinscrire d'activités via des points de terminaison dédiés.
